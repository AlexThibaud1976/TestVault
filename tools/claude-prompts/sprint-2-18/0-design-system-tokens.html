Typography

Segoe UI system font (Q6=a). Scale: 11/12/14/16/20/24/32. Weight 400/600.

--t-display Test Plans 32px / 600
--t-h1 Plan section title 24px / 600
--t-h2 Subsection heading 20px / 600
--t-h3 Card or panel title 16px / 600
--t-body Body text used for descriptions and main content. 14px / 400
--t-small Helper, metadata, secondary labels 12px / 400
--t-micro OVERLINE / TAGS 11px / 600
